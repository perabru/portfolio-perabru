# Portfólio Black — Bruno Michel Pera

Portfólio profissional em tela preta, com navbar blur, animações, botão de WhatsApp, Currículo Lattes e projetos organizados em menu expansível.

## Funcionalidades

- Interface preta, limpa e profissional.
- Navbar com blur.
- Animações de entrada.
- Efeito de luz acompanhando o mouse.
- Cards com hover.
- Botão fixo para WhatsApp.
- Link para Currículo Lattes.
- Projetos organizados em menu expansível por categoria.
- Busca por repositório.
- Ordenação por atualização, nome ou estrelas.
- Dados públicos carregados automaticamente do GitHub.

## Como usar

Abra o arquivo `index.html` no navegador.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie os arquivos `index.html`, `style.css`, `script.js` e `README.md`.
3. Acesse `Settings`.
4. Entre em `Pages`.
5. Em `Source`, selecione a branch principal.
6. Aguarde o link público do GitHub Pages.
